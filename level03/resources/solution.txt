1. Найти в домашней директории исполняемый файл
2. Запустить файл level03 с трассировкой стека программой ltrace
3. Увидеть, что программа делает системный вызов echo для вывода
4. Создать исполняемый файл с вызовом getflag и назвать его echo
5. Изменить переменную PATH так, чтобы поиск команды echo начинался с этого файла
6. Взять флаг из вывода level03
