1. Найти в домашней директории исполняемый файл
2. Узнать порт, запустив файл level10 с валидными аргументами
3. Узнать, что программа использует системный вызов access для проверки доступа к файлу
4. Создать ссылку на файл token, создать пустой файл
5. Запустить netcat для просмотра переданных запросов
6. Менять в цикле ссылку между файлом token и доступным файлом
7. Запускать в цикле файл level10, пока проверка доступа и открытие файла не попадут между сменой ссылки
8. Найти пароль в выводе netcat
9. Взять флаг из вывода getflag с этим паролем
